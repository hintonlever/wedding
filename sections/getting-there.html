Getting There

We appreciate some of you will be coming a very long way!

The closest airport is in Rockhampton, with multiple flights per day from Brisbane. From the airport, it's a 50-minute drive to the marina in Yeppoon. There are taxis but no ubers.

$55pp return on Keppel Connections
$60pp return on Freedom Fast Cats

Once on the island everything is walkable.

Please note it is not possible to fly Brisbane to Rockhampton on Saturday morning and make it in time.
